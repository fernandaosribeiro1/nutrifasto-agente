# 🥗 NutriTrack AI

Aplicação web moderna para acompanhamento nutricional, controle de jejum intermitente e monitoramento de metas diárias.

O projeto foi desenvolvido com foco em experiência do usuário, responsividade e integração com Inteligência Artificial para auxílio nutricional.

---

## ✨ Funcionalidades

- ✅ Controle de refeições diárias
- ✅ Contagem de calorias
- ✅ Controle de ingestão de água
- ✅ Sistema de jejum intermitente
- ✅ Histórico de jejuns
- ✅ Dashboard com gráficos
- ✅ Planner semanal
- ✅ Integração com IA nutricional
- ✅ Autenticação com Firebase
- ✅ Interface responsiva e moderna

## ⚙️ Como rodar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/fernandaosribeirodev/nutritrack-agente.git
```

### 2. Acesse a pasta

```bash
cd nutritrack-agente
```

### 3. Instale as dependências

```bash
npm install
```

### 4. Configure o arquivo `.env.local`

```env
NEXT_PUBLIC_FIREBASE_API_KEY=
NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN=
NEXT_PUBLIC_FIREBASE_PROJECT_ID=
NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET=
NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
NEXT_PUBLIC_FIREBASE_APP_ID=

OPENROUTER_API_KEY=
```

### 5. Rode o projeto

```bash
npm run dev
```
